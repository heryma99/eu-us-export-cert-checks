皮具/箱包专项认证

适用于女包（真皮/PU/帆布）、手提袋、背包、钱包等皮革及非皮革箱包制品
LWG 皮革工作组认证
P1 市场强制（准入门槛） 皮具
Leather Working Group — 全球最权威的皮革行业环境与社会责任审核。耐克、开云集团、宜家等品牌将LWG认证设为供应商准入门槛。2025年重大变化：审核权由品牌回收至LWG总部统一管理，门槛再升级。评级分金/银/铜三级，银级及以上为多数品牌最低要求。
核心理由真皮女包出口欧盟，欧洲品牌商几乎100%要求供应商通过LWG
适用范围皮革鞣制/加工企业，及皮革制品品牌商（可追溯认证）
周期/费用审核周期3-6个月，年费约€2,000-5,000 + 审核费
行动项确认皮革供应商是否已取得LWG，若未取得则启动申请
箱包物理性能检测
P1 买家合同要求 皮具
包括：皮革色牢度（摩擦/光照/汗渍）、五金件耐腐蚀（盐雾测试）、拉链/扣件耐久性、缝合强度、承重测试。虽无法定强制，但欧美品牌商/零售商均以合同条款要求提供第三方检测报告。
关键标准EN ISO 17075（六价铬）、ISO 11640（色牢度）、QB/T 1333（背提包）
检测机构SGS / Intertek / BV / TÜV，出英文报告
周期5-10个工作日/批次
行动项每季度送检1-2款代表性产品，建立检测档案库
CITES 濒危物种贸易公约
P1 1975年 已执行（法律强制） 皮具
若产品使用鳄鱼皮、蛇皮、蜥蜴皮等濒危动物皮革，须取得CITES进出口许可证。无证进出口即构成走私，在欧美海关可被扣押销毁并追究刑事责任。
触发条件涉及CITES附录I/II物种皮革（蟒蛇、鳄鱼、巨蜥等）
出口文件中国出口许可证 + 进口国进口许可证（双证齐全）
风险极高 — 欧美海关对此类产品有专门稽查通道
行动项若产品不使用珍稀皮革，在商业发票注明"All materials are non-CITES"
包袋五金件无镍认证
P1 EU REACH 已执行 皮具
EU REACH附录XVII第27条：与皮肤长期接触的金属部件（拉链头、扣件、铆钉、链条），镍释放量须 < 0.5 μg/cm²/week。超标将触发REACH不合规。此要求对女包五金件装饰性强的产品尤其关键。
检测标准EN 1811（镍释放量）、EN 12472（磨损模拟 + 镍释放）
适用部件拉链头、金属扣、铆钉、链条、磁吸扣、D环
周期5-7个工作日
行动项要求五金供应商提供无镍声明+检测报告，每批次抽检

鞋类专项认证

适用于女鞋（皮鞋/运动鞋/凉鞋/靴）、拖鞋等鞋类制品
SATRA 鞋类检测与认证
P2 自愿性认证 鞋
SATRA（英国鞋业研究协会）是全球鞋类检测与认证的权威机构。提供鞋类物理性能全套检测：鞋底耐磨、粘合强度、防水性、弯曲疲劳、鞋跟抗冲击。虽非法定强制，但欧美大型鞋类零售商（如Clarks、ECCO等）普遍要求供应商通过SATRA检测。
核心标准SATRA TM144（鞋底耐磨）、SATRA TM60（粘合强度）、EN ISO 20344（安全鞋通用）
适用范围所有出口欧美的鞋类产品，尤其是功能性鞋（防水/户外）
周期/费用检测2-4周，费用约£500-1,500/款（依检测项目数）
行动项若客户未指定SATRA，可先用SGS/Intertek等同级检测替代
安全鞋 CE 认证（PPE法规）
P3 EU 2016/425 已执行 鞋（安全鞋）
若出口安全鞋/劳保鞋/防护鞋（钢头鞋、防静电鞋、防穿刺鞋等），必须通过EU PPE法规2016/425的CE认证，由欧盟公告机构（如SATRA）出具EU型式检验证书。普通时尚女鞋不涉及此项。
触发条件产品宣称具有安全/防护功能（钢头、防砸、防静电等）
分类SB/S1/S2/S3（安全级）、PB/P1/P2/P3（防护级）
周期/费用型式检验3-6个月，费用€3,000-8,000
行动项如贵司不涉及安全鞋品类，此项为N/A，确认后勾选
鞋类防滑性能检测
P1 买家合同/保险要求 鞋
美国市场尤其重视鞋底防滑性能——若消费者因滑倒受伤，零售商/品牌方可能被起诉。多数美国零售买家将防滑检测报告列为合同必备文件。
关键标准ASTM F2913（鞋底摩擦系数）、SATRA TM144、EN ISO 13287
检测条件干态 + 湿态双条件测试
周期5-7个工作日
行动项每季度送检1-2款代表性鞋款，出ASTM/EN双语报告
鞋材有害物质检测（DMF/偶氮等）
P1 EU REACH 已执行 鞋
鞋类特有的化学风险：防霉片中的DMF（富马酸二甲酯，2009年EU明文禁止）、鞋底/鞋面材料中的多环芳烃(PAHs)、含氯酚（防霉剂）。每项均有具体的REACH限量要求。
DMFEU条例(EU) 2013/135，检出即不合格，禁止销售
PAHsREACH Annex XVII 第50条，8项PAHs总量≤限值
周期5-7个工作日
行动项要求鞋材供应商提供REACH合规声明，出厂前DMF必检

服装/纺织专项认证

适用于女装（棉/麻/化纤/混纺）、连衣裙、上衣、裤装等服装制品
GOTS 全球有机纺织品标准 (v7.0)
P2 自愿性认证 服装
Global Organic Textile Standard — 有机纺织品领域最高级别认证。V7.0于2023年发布，2024年3月起全面实施。要求产品含≥70%有机天然纤维，且全供应链通过GOTS审核（从轧棉→纺纱→织造→染整→成衣）。标注"organic"的产品须≥95%有机纤维。
适用场景若贵司有"有机棉"女装系列，GOTS是欧美高端市场的核心卖点
门槛≥70%有机天然纤维，禁用GMO、有毒化学品、童工
周期/费用认证周期6-12个月，年费€1,500-4,000 + 审核费
行动项评估有机棉原料供应可行性，非核心品类则暂缓
OCS 有机含量标准
P3 自愿性认证 服装
Organic Content Standard — Textile Exchange推出的有机纤维含量追踪标准。仅在含量层面认证（5-100%有机纤维），不涉及化学品/环境/社会责任。比GOTS门槛低，适合有机棉混纺产品。分OCS 100（≥95%）和OCS Blended（≥5%）。
与GOTS区别OCS仅追踪纤维含量，GOTS覆盖全流程（化学品+社会+环境）
适用场景有机棉混纺产品（如50%有机棉+50%涤纶），无法达标GOTS
周期/费用3-6个月，费用低于GOTS约30-40%
行动项如果做有机概念但无法全链路合规，OCS是折中方案
BCI 良好棉花倡议
P3 自愿性认证 服装
Better Cotton Initiative — 全球最大的棉花可持续发展项目。不是有机棉认证，而是推动棉花种植的水/农药/体面劳动的持续改进。H&M、Nike、IKEA等品牌均为BCI会员，要求供应商采购BCI棉花。
性质非产品认证，而是供应链体系认证（mass balance可追溯）
适用场景常规棉产品，品牌客户要求"可持续棉花"来源
成本极低 — BCI棉花与常规棉花价差很小（约1-3%溢价）
行动项若品牌客户要求，可要求面料供应商使用BCI认证棉花
RDS 责任羽绒标准
P3 自愿性认证 服装（羽绒）
Responsible Down Standard — Textile Exchange推出的羽绒溯源标准，禁止活拔羽毛和强制喂食。若贵司产品含羽绒填充（羽绒服/羽绒背心），欧美品牌客户几乎100%要求RDS认证。
触发条件产品含有任何羽绒/羽毛填充料
核心要求禁止活拔、禁止强制喂食、全链条可追溯至养殖场
周期3-6个月（供应链审核）
行动项与羽绒供应商确认是否已取得RDS，或启动新供应商开发
RWS 责任羊毛标准
P3 自愿性认证 服装（羊毛）
Responsible Wool Standard — Textile Exchange推出的羊毛动物福利标准。若贵司产品含羊毛面料（羊毛大衣/毛衣），欧美高端品牌客户（如Patagonia、H&M集团）要求RWS认证。
触发条件产品含有羊毛成分 ≥5%
核心要求动物福利五项自由、土地可持续管理、全链条可追溯
周期3-6个月
行动项若产品不含羊毛，此项不适用；如有羊毛产品，与面料供应商核实

工厂社会责任审核

欧美品牌商/零售商对供应商工厂的强制性审核要求，覆盖劳动权益、健康安全、环境管理
BSCI 倡议商界遵守社会责任组织审核
P1 市场强制（品牌统一要求） 工厂
amfori BSCI（原BSCI）是全球最广泛采用的社会责任审核体系之一，覆盖3,000+品牌会员。审核涵盖13大绩效领域：不歧视、公平报酬、体面劳动时间、职业健康安全、不雇佣童工、不强制劳动、结社自由等。欧美零售商/品牌方通常要求供应商持有有效的BSCI审核报告（有效期2年）。
评级A/B/C/D/E五级，多数品牌要求C级及以上
审核周期通知审核或半通知审核，2-4天现场 + 2周出报告
费用¥15,000-30,000/次（依工厂规模），2年有效期
行动项确认合作工厂是否持有有效BSCI报告，若未有则启动审核
SEDEX / SMETA 6.1 审核
P1 市场强制（品牌统一要求） 工厂
SEDEX（供应商道德数据交换）是全球最大的供应商道德信息共享平台。SMETA是其官方审核方法论，最新版为SMETA 7.0（2025年发布），升级为4-Pillar审核：劳工标准 + 健康安全 + 环境评估 + 商业道德。与BSCI互为替代，多数品牌接受BSCI或SMETA之一。
2026年更新SMETA 7.0强制增加突击审核维度、供应链透明度要求
与BSCI关系多数品牌二者选一即可，部分品牌（如沃尔玛）要求SMETA
费用¥12,000-25,000/次，平台会员年费另计
行动项与主要品牌客户确认：BSCI还是SMETA？按客户指定推进
WRAP 全球责任服装生产认证
P2 自愿性（美国市场偏重） 工厂
Worldwide Responsible Accredited Production — 专门针对缝制品（服装/鞋/箱包）行业的社会责任认证。1997年成立，认证工厂覆盖全球。美国市场（尤其是大型零售商）对WRAP的认可度高于欧洲。
认证级别白金/金/银三级，白金级为最高（需3次连续合规审核）
12项原则包括合法用工、禁止强迫劳动、禁止童工、反骚扰、薪酬工时、健康安全等
周期初次认证3-6个月，年度监督审核
行动项若美国大型零售商（如Target/Walmart）是重点客户，优先WRAP
SA8000 社会责任标准
P3 自愿性（高端品牌要求） 工厂
Social Accountability International推出的全球首个社会责任认证标准（1997年）。比BSCI/SMETA更严格：要求企业建立工人参与的社会责任管理体系，审核更深入。某些欧洲高端品牌（如Gucci母公司开云集团）倾向要求SA8000。
核心差异SA8000是管理体系认证（需建立制度），BSCI/SMETA是绩效审核
适用场景欧洲奢侈品牌供应商、强调ESG的长期合作项目
周期/费用6-12个月，首次认证约¥50,000-80,000
行动项90欧均价产品定位下，BSCI/SMETA已足够，SA8000暂缓
SLCP 社会劳工融合项目
P2 新兴标准 工厂
Social & Labor Convergence Program — 旨在减少重复审核的行业融合项目（2019年启动）。工厂完成一次SLCP评估后，数据可被多个品牌/标准方共享（BSCI、Higg FSLM等均接受SLCP数据）。2026年越来越多品牌要求供应商先完成SLCP自评。
核心价值一次评估，多方共享 — 避免重复审核，大幅降低审核成本
流程工厂自评 → 第三方验证 → 数据上传Gateway → 品牌共享
费用验证费¥8,000-15,000（远低于完整审核）
行动项若面对多个品牌客户的重复审核，优先推进SLCP一劳永逸

环境管理与可持续发展认证

覆盖化学品管理、碳足迹、环保体系 — 欧美买家的ESG合规要求持续升级
ZDHC 有害化学物质零排放
P0 市场强制（品牌联合要求） 工厂
Zero Discharge of Hazardous Chemicals — 由Nike、Adidas、H&M、Levi's等30+全球品牌发起。要求供应链工厂在生产过程中消除有害化学物质排放。ZDHC MRSL（制造过程受限物质清单）是行业硬标准，2026年版本持续扩充。品牌客户要求供应商在ZDHC Gateway平台上传废水/污泥检测数据。
核心要求废水检测（ZDHC废水指南）、化学品清单审核（MRSL合规）
2026更新MRSL v3.2发布，新增PFAS等限制物质
适用范围染整/印花/水洗等湿处理工序的工厂，纯贸易商不直接涉及
行动项要求染厂/面料供应商提供ZDHC合规声明与废水检测报告
bluesign® 蓝标认证
P2 自愿性（高端市场） 服装
瑞士bluesign technologies推出的化学品与环保审核体系。从输入端控制有害物质（比OEKO-TEX更严格），覆盖化学品、水资源、废气排放、职业健康、消费者安全五大维度。户外/运动品牌（Patagonia、The North Face、Mammut）普遍采用。
与OEKO-TEX区别bluesign从生产端输入控制，OEKO-TEX从成品端检测
认证对象化学品生产商、面料厂、辅料厂、品牌商（全链条）
周期/费用系统审核6-12个月，年费€5,000-15,000 + 审核费
行动项若户外/运动品牌客户有此要求，推动面料供应商取得bluesign
Higg Index FEM 工厂环境模块
P1 市场强制（品牌要求） 工厂
Higg Facility Environmental Module — 可持续服装联盟(SAC)推出的工厂环境绩效自评工具。覆盖环境管理体系、能源/温室气体、水资源、废水、废气、固废、化学品七大维度。200+品牌（含Nike、H&M、Patagonia等）要求供应商完成Higg FEM自评并在平台上公开分数。
2026更新Higg FEM 4.0版本，增加碳减排目标设定、物理风险暴露评估
流程工厂在线自评 → 第三方验证（选做）→ 品牌在Higg平台查看
周期/费用自评1-3个月，验证费$2,000-5,000/年
行动项若品牌客户要求，在Higg.org注册并完成FEM自评
ISO 14001 环境管理体系
P2 自愿性 工厂
国际标准化组织环境管理体系标准，最新版为ISO 14001:2015。虽非法定强制，但越来越多的欧美品牌将ISO 14001列为供应商准入门槛条件，尤其是在ESG合规审查中。
核心内容环境方针、环境因素识别、合规义务、目标与方案、内审、管理评审
周期初次认证6-12个月，3年有效期，每年监督审核
费用¥15,000-40,000/初次 + ¥8,000-15,000/年监督
行动项如工厂已有ISO 9001，可同机构增项ISO 14001降低成本
ISO 45001 职业健康安全管理体系
P2 自愿性 工厂
替代OHSAS 18001的职业健康安全管理体系国际标准（2018年发布）。与ISO 14001结构一致（Annex SL框架），可与ISO 9001/14001整合认证。欧美品牌ESG审核时视为加分项。
核心内容危险源辨识、风险评估、法律法规合规、应急准备、员工参与
整合优势可与ISO 9001+14001整合认证（IMS），节省审核成本约30%
费用单独认证¥15,000-35,000/初次
行动项升级OHSAS 18001至ISO 45001（如已有旧标）；或新增
FSC 森林管理委员会（包装）
P3 自愿性 全品类
若产品包装（吊牌、鞋盒、包装纸）使用纸质材料并有"环保"宣称，FSC认证可证明木材/纸浆来源合法且可持续。欧盟2023年已生效的《零毁林法案》(EUDR)进一步提高了对林产品溯源的要求。
适用对象包装供应商（印刷厂/制盒厂），而非产品本身

贵司行动优先级建议

基于"女包+鞋服、均价90欧、FBA+传统贸易混合"模式

阶段一：2026年7月前（合规生存）

CPSC eFiling送检支撑（GCC+16 CFR 1610）+ GPSR欧代标签合规（已在基础合规清单中覆盖）。工厂社会责任审核：启动BSCI或SMETA，选其一——优先向主要品牌客户确认他们接受哪一个。

阶段二：2026年Q3-Q4（竞争力提升）

LWG认证推进（如真皮女包是核心品类）、Higg FEM自评（如品牌客户已要求）、ZDHC合规（要求面料/染厂供应商提供废水检测数据）。

阶段三：2027年（品牌溢价）

按客户需求选择性推进：GOTS/OCS（如启动有机棉系列）、bluesign（如户外/运动品牌客户增加）、GRS（如使用回收材料）、SLCP（一劳永逸减少重复审核）。

关键提醒

① 社会责任审核 + 环境认证的主体是工厂，贵司作为贸易公司，核心任务是确保合作工厂持有有效证书。
② 切勿混淆"产品检测"与"工厂审核"——前者是每批次/每款产品送检，后者是工厂年度审核。
③ 三大社会责任审核（BSCI/SMETA/WRAP）选其一即可，不必全做——向主要客户确认偏好。
